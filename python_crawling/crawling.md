    <h2>1. 원리</h2>
    <p>(1) 웹 드라이버란 ?
    - 웹 문서를 분석하고 이를 활용하여 화면 구성
    - 웹 문서에 이벤트를 전달하고 결과값을 받음
    - 웹 드라이버가 제공하는 방법으로 서로 주고 받아야 함</p>
    <p>(2) selenium 라이브러리 사용
    - 일종의 서버 프로그램인데 라이브러리로 제공
    - 다양한 브라우저의 웹 드라이버 컨트롤 가능</p>
